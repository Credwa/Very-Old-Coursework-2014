Skylace System Requirements
If the system doesn't meet the requirements of the game, the game will either not run at all or give a less than desirable performance. To ensure a smooth gaming experience, we recommend that you meet the following settings:Operating System:

Windows Vista, Windows 7, Windows 8, Mac

CPU:

Intel Core i7 -3630QM clocked at 2.4GHz, (Mid to High Range Dual-Core 2.5GHz CPU or better for minimum.)

RAM:

6GB of RAM (8GB or more for Windows Vista and Windows 7)

Hard Disk:

30GB Free HD Space

Graphic Display:

3D video chipset with pixel and vertex shader model 1.3 supported. For Windows Vista, 7, and 8 a NVIDIA Gefore GT 635M 2gb Ram or newer is needed.

Sound Drivers:

DirectX 11.0c compatible sound card

Network:

Broadband Internet connection
